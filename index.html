Portal

もなとれ (mona-tore)

モナーコイン（Monacoin; MONA）を基軸通貨とする世界初の暗号通貨取引所。モナーコイン、ビットコイン（Bitcoin; BTC）、ライトコイン（Litecoin; LTC）、クマコイン（Kumacoin; KUMA）、Ringo（RIN）など、国産暗号通貨を中心として多数の通貨に対応。

もなぞん。

モナーコインでAmazonギフトコードを購入できるサイト。ユーザ登録からギフトコード発送まで全自動なので素早く購入できるのが特色。

モナーコイン Debian/Ubuntu パッケージ

Monacoin のクライアント (monacoin-qt, monacoind) の Debian/Ubuntu パッケージを作るために必要なファイル群。

Ubuntu をお使いの方は PPA (Personal Package Archive; 個人でリポジトリを作ることのできるサービス) にパッケージをアップロードしましたので、以下のコマンドを打つだけで Monacoin を導入することができます。

$ sudo add-apt-repository ppa:monatr/monacoin
$ sudo apt-get update
$ sudo apt-get install monacoin-qt monacoind
